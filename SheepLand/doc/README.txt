HOW TO PLAY

-------------------------------------------------------------------------
1)Start the server
1.1) Open a console and navigate to the folder of "sheepland.jar"
1.2) if LINUX execute "sheepland.jar -s"
   if WINDOWS execute "java -jar sheepland.jar -s"
   if MAC execute java -jar sheepland.jar -s"
1.2) the server doesn't receive command, 
   WARNING: once the server is shutdown all the data about game and players will be lost
-------------------------------------------------------------------------
2)Start a client
2.1) Open another console and navigate to the folder of "sheepland.jar"
2.2) if LINUX execute "sheepland.jar -c"
   if WINDOWS execute "java -jar sheepland.jar -c"
   if MAC  execute "java -jar sheepland.jar -c"
2.3) Follow the instuctions
-------------------------------------------------------------------------
3)To start a game open 2 or 3 or 4 clients
When the fourth gamer make the login the game starts.
After the second gamer make the login a timer start and the game will start 
within n sec unless the fourth one arrives.

PS: RMI and socket players can play together
